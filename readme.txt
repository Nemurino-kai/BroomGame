ほうきを倒さないげーむ

作者:DoubleZ(だぶぜ)
_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/
『ほうきを倒さないげーむ』は、ウィンドウを左右に移動させることにより、
手に乗せた箒を倒さないよう、バランスを取るゲームです。
飛んでくる障害物に耐え、箒を立てられた時間を競います。

-はじめる
ゲームをはじめからやり直します。

-とくてん
箒を立てた時間の『ランキング』を見ることができます。

-くれじっと
素材の提供元などを表示します。

-おわり
ゲームを終了します。
